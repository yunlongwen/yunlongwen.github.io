队列介绍

队列是一个有序列表，可以用数组或是链表来实现。
遵循先入先出的原则。即：先存入队列的数据，要先取出。后存入的要后取出

数组实现

使用数组实现一个循环队列，整理思路如下

  • 定义一个环形相关的变量
  • front 表示队列第一个元素的下标
  • rear 表示队列最后一个元素的下标
  • maxSize表示队列的最大容量
  • currentSize表示当 前队列的有效数据个数
  • 用定义的变量来计算出队列的状态
  • 队列是否为空
    front == rear
  • 队列是否满
    让front和 rear 相邻时，且（首尾相连），表示队列已经满了
    （rear - front）%（maxSize - 1） == 0
  • 队列的有效数据
    前提是不为空，即front != rear
    rear % maxSize - front % maxSize +1
  • 加入队列
    array[rear]= data
    rear = (rear+1) % maxSize
  • 从队列取出
    var data = array[front]
    front = (front+1) % maxSize
    return data
